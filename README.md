Web-based Deep Learning Model Build framework  
This project was created in order to learn deep learning intuitively

## Install Dependency

---

```
$ npm i
   or
$npm install
```

## Build

---

```
$ npm build
```

## Start framework

---

```
$ npm run serve
```
